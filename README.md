# Signal to Shell

Signal to Shell is a web-based, interactive educational platform designed to take users from terminal basics to mastering the Gemini CLI. The curriculum is divided into five deployment phases (Blocks), each inspired by Tame Impala's discography, guiding users through file system manipulation, logic patterns, and advanced agentic orchestration.

## Features
- **Virtual File System (VFS):** A browser-based file system emulator with command persistence.
- **Agentic Curriculum:** 50+ lessons covering everything from `pwd` to autonomous self-healing state architectures.
- **Cyberpunk Aesthetic:** A high-fidelity "Netrunner Nexus" design system.
- **MCP Integration:** Hands-on experience with the Model Context Protocol.

## Getting Started
First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to begin the curriculum.

## Testing
The project includes a comprehensive Playwright test suite covering all curriculum blocks.

```bash
npx playwright test
```
